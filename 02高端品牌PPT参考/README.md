## 设计参考要点

1. **色彩体系**：奶油白/米白 (#F2EDE6)、黑金、深蓝金等高端配色
2. **排版风格**：大字号标题、细线装饰、留白充足
3. **图片运用**：全出血大图、产品特写、人物生活方式照片
4. **布局特点**：非卡片化、多层级、电影感构图
5. **装饰元素**：细线框、金色点缀、几何图形
