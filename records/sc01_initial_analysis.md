# sc_01_white_gold_project_proposal 初始分析

## slide_01

这是一页典型的高端金属质感封面页。背景不是纯渐变，而是带有镜面反射与弧面折叠感的金棕色抽象材质，右侧有一条从右上到中下的大型弧形高光边缘，底部左侧也有折角般的浅色高光面。标题采用极高对比的白色超大号压缩无衬线字，左对齐，分成两行：`ELEGANT` 与 `PROJECT PROPOSAL.`。标题区域位于左上到中左位置，小副标题 `A PRESENTATION` 在标题下方，字距较松。右上角是很小的页眉文本 `WHITE & GOLD PROJECT PROPOSAL`。底部有一条细白横线贯穿大部分宽度，左下和右下分别放作者与日期。

可复用组件包括：

1. 金属流线背景层，可用多层径向渐变、线性渐变与弧形遮罩模拟。
2. 左侧超大压缩标题组件。
3. 右上页眉组件。
4. 底部细分隔线与左右信息栏组件。

## slide_02

这是说明/目录式信息页。整体背景为极浅灰白色，左上与右下各有一块金色流体纹理装饰，边缘带细线描边感。主标题为黑色超大压缩无衬线字 `HOW TO USE THIS PRESENTATION.`，末尾句点是金黄色强调。正文区域是三栏结构，分别为 `GOOGLE SLIDES`、`POWERPOINT`、`CANVA`，栏标题使用同款压缩无衬线黑体。正文为黑色圆点列表与段落文字，左栏和中栏内容较多，右栏较少。右上角仍保留小页眉。底部保留细黑横线，左下和右下继续显示作者与日期。

可复用组件包括：

1. 浅底页面框架。
2. 金色流体角落装饰图层。
3. 超大左对齐标题组件（支持末尾彩色句点）。
4. 三栏信息区组件。
5. 顶部页眉与底部页脚组件。

## 后续复刻策略

优先先完成 sc_01 全模板，因为其风格统一、组件可复用性最高。建议先建立一个共享样式框架：

- `corner-marble-gold`：说明页与内容页的角落纹理装饰。
- `three-column-copy`：三栏说明正文布局。

后续继续查看 sc_01 其余页面，再基于复用组件批量编写 HTML。

## slide_03

这是目录页变体。左侧约占三分之二宽度，使用超大标题 `TABLE OF CONTENTS.`，其下有一条短横线。正文为两列四项的目录块，每块由压缩无衬线小标题与两行说明文组成。右侧三分之一是贯穿全高的玛瑙/矿石纹理装饰图，色彩从白、金、棕到深棕层层叠加，边缘呈不规则波纹。底部仍是统一页脚线与作者/日期信息。

可复用模式：

- 标题下横线。
- 二列目录块卡位系统。

## slide_04

这是章节过渡页。整体采用暖金褐色柔焦背景，右下保留一个弧形高光边缘体块。页面几乎无正文，只在中央放置巨大白色标题 `INTRODUCTION.`，右上保留页眉，底部保留页脚线与信息。该页与封面不同，标题居中，整体更像章节封面。
